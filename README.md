# E-Commerce Sales Analysis Dashboard

## Project Overview

This project analyzes e-commerce sales data using SQL and Power BI to identify sales trends, profitability, top-performing products, and regional performance.

## Tools Used

- SQL
- MySQL
- Power BI

## Dashboard

![Dashboard](images/dashboard.png)

## Key Metrics

| Metric | Value |
|----------|----------|
| Total Sales | $2.27M |
| Total Profit | $282.86K |
| Total Orders | 5K |
| Total Quantity | 37K |
| Profit Margin | 12.45% |

## SQL Analysis Performed

- Total Sales Analysis
- Total Profit Analysis
- Sales by Category
- Profit by Category
- Sales by Region
- Top Products Analysis
- Top Customers Analysis

## Business Insights

- Total Sales reached $2.27M.
- Total Profit reached $282.86K.
- Technology generated the highest sales.
- West region contributed the largest share of revenue.
- The business maintained a profit margin of 12.45%.

## Dashboard Features

- KPI Cards
- Sales by Category
- Profit by Category
- Sales by Region
- Monthly Sales Trend
- Top Products Analysis
- Interactive Slicers
